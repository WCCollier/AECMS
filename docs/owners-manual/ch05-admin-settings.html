Chapter 5 — Admin Settings

A reference for every settings tab in the backstage.

Full guide coming soon — key sections documented below.

General

Site title, tagline, timezone, date format, and homepage mode. The Homepage setting controls what visitors see at your root URL (/):

  • Article Feed — redirects / to /articles
  • Custom Page — shows a specific page at /. Select a published page from the dropdown. If the selected page is not published, the site falls back to the _home_ page, then to the article feed.

Email / SMTP

Enter your SMTP credentials here. Required fields: Host, Port, Username, Password, From Address. Click Test to send a test email using the values currently in the form (you do not need to save first). Save after a successful test.

Payment Providers

Stripe and PayPal credentials. Click Verify after entering credentials to confirm the API connection is working. Stripe handles cards, Apple Pay, Google Pay, and Amazon Pay. PayPal is an alternative checkout option.

File Storage

Shows the active storage provider and bucket configuration. Fields set via environment variables appear with an env var badge and are read-only — to change them, update the environment variable and redeploy. You can override env var values by entering a value here; the in-database value takes precedence.

Click Test Storage Connection to verify a write/read/delete round-trip against your storage provider using the values currently in the form.

⚠️ Changing bucket names after files are uploaded

If you change the media or digital bucket name after files have already been uploaded, existing media URLs and download links will break. You must migrate files to the new bucket before changing the name here.

Appearance

Choose a color palette and font pairing. Changes apply site-wide immediately after saving. Available palettes: Midnight, Ocean, Forest, Sunset, Rose, Storm, Ember, Arctic.
