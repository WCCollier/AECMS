Place the following Kindle guide screenshots in this directory:

  to-kindle_1_marked.jpg  — Amazon.com Account page, green oval on "Devices" link
  to-kindle_2_marked.jpg  — Amazon Devices page, green oval on "Kindle" category
  to-kindle_3_marked.jpg  — Kindle device detail, green oval on "Preferences" tab
  to-kindle_4_marked.jpg  — Preferences / Personal Document Settings, magenta oval on "Add a new e-mail address"

The KindleWizard component (frontend/components/digital/KindleWizard.tsx) loads these
automatically. If any image is missing it degrades gracefully (shows placeholder text).

Recommended: 1280×800px, JPEG quality 80. The images will be cropped to a 16:9 aspect ratio
in the carousel.
