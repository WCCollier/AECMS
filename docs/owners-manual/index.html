AECMS Owner's Manual

Everything you need to deploy, configure, and run your own CMS and online store — from choosing a host to publishing your first article.

What is AECMS?

AECMS (Advanced Ecommerce Content Management System) is a self-hosted CMS and ecommerce platform. You own the code, the database, and the data. There is no monthly SaaS subscription — you pay only for the hosting services you choose.

It is designed for personal websites and small online stores with low to moderate traffic. It runs in Docker containers, so it can be deployed to almost any hosting provider.

How This Manual Is Organized

Setup happens in four stages. Each stage has its own chapter.

1 — Choose Infrastructure
Pick a hosting platform, database, file storage provider, and email service before you install anything.
2 — Platform Setup
Step-by-step instructions for your chosen host: Railway, Coolify, Cloud Run, Render, or Fly.io.
3 — Configure AECMS
Set the environment variables that tell AECMS which services to use before you deploy.
4 — First Launch
Run the setup wizard, confirm your storage and email connections, and connect your domain.

Quick-Start Paths

Not sure where to begin? Pick the path that matches your situation:

I want the cheapest possible option

→ Coolify on a $5/month VPS. Coolify is free software that runs on a virtual server you rent from Hetzner, DigitalOcean, or Linode. You get PostgreSQL, Redis, and file storage all on the same server. Total cost: ~$5–6/month.

I want the easiest possible setup with no command line

→ Railway. Connect your GitHub account, import the repo, and Railway handles builds, database, Redis, and deployments entirely through a web UI. Starting at $5/month for a hobby plan.

I want production-grade infrastructure that can scale

→ Google Cloud Run. Containerized, serverless, auto-scaling. PostgreSQL via Cloud SQL, Redis via Upstash, files on Google Cloud Storage. More setup steps, but the most robust option. ~$10–12/month at low traffic.

What You Will Need

  • A GitHub account (to fork the repository)
  • An account on your chosen hosting platform
  • A domain name (optional for testing, required for a live site)
  • Approximately 1–2 hours for a first-time install
ℹ️ No programming required

You do not need to write any code to install AECMS. You will copy-paste some configuration values and click buttons in web dashboards. The platform guides in Chapter 2 walk through every step.

Before You Start

Read Chapter 1 before doing anything else. It will help you decide which hosting platform and supporting services to use. Making these decisions upfront takes 10 minutes and saves you from having to redo the install.
