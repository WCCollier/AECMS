Deploying to Fly.io

Generous free tier. Requires Fly CLI.

Cost: ~$0–10/month depending on usage

ℹ️ Guide coming soon

The full Fly.io setup guide is in progress. Key notes for early adopters:

  • Install the Fly CLI: curl -L https://fly.io/install.sh | sh
  • Deploy the backend: fly launch --dockerfile backend/Dockerfile from the repo root
  • Deploy the frontend: fly launch --dockerfile frontend/Dockerfile
  • Use fly postgres create for the database and Upstash for Redis (free tier)
  • Set environment variables with fly secrets set KEY=VALUE
  • Add your domain: fly certs add yourdomain.com
  • Starter config files are in deploy/fly/
